# Grid Row Detail

<cds-alert-group status="warning" cds-layout="m-b:lg">
  <cds-alert>
    This component or utility is offered as a preview. This means we are currently working on it and seeking feedback.
    Please be aware that this component or utility may have breaking changes before we finish working on it.
  </cds-alert>
</cds-alert-group>

The `cds-grid-detail` component provides a modal-like overlay to display additional details to a particular row entry.
A `cds-grid-detail` can be visually toggled with `hidden`. To show the `cds-grid-detail` toggle its visibility using
`cds-button-action` in a given grid row. Detail can be positioned left, right or bottom of the datagrid with the `position` property.
By default it is positioned right. Left can be used for RTL support and bottom for expand style rows.

The row detail will auto align to any cell with `role="rowheader"` as an identifier when positioned to the left or right.
If no `role="rowheader"` is found it will default to `70%` of the grid. The fallback width can be cusomized via
the `--width` CSS Custom Property.

<Story id="stories-grid--row-detail" />

```html
<cds-grid>
  <cds-grid-column></cds-grid-column>
  <cds-grid-column>Host</cds-grid-column>
  <cds-grid-column>Status</cds-grid-column>
  <cds-grid-column>CPU</cds-grid-column>
  <cds-grid-column>Memory</cds-grid-column>
  <cds-grid-row>
    <cds-grid-cell>
      <cds-button-expand action="detail" id="current-detail-demo" aria-label="view host details"></cds-button-expand>
    </cds-grid-cell>
    <cds-grid-cell role="rowheader">vm-host-001</cds-grid-cell>
    <cds-grid-cell>online</cds-grid-cell>
    <cds-grid-cell>5%</cds-grid-cell>
    <cds-grid-cell>10%</cds-grid-cell>
  </cds-grid-row>
  <cds-grid-detail anchor="current-detail-demo">
    <h2>Host: vm-host-001</h2>
    <p>Status: online</p>
    <p>CPU: 5%</p>
    <p>Memory: 10%</p>
  </cds-grid-detail>
</cds-grid>
```
